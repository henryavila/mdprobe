Como mostrar o anchor status das annotations?

Quando drift é detectado, cada annotation é re-ancorada e classificada. Como surfacear isso na UI?

A

Status Badges nos Cards

Cada annotation card no painel direito ganha um badge colorido indicando o status de ancoragem

Pros

  • Status visível sem precisar interagir
  • Cores da border-left mudam conforme status
  • Compatível com filtros existentes

Cons

  • Orphans ficam misturadas com anchored
  • Com 20+ annotations, difícil achar as problemáticas
B

Seção Separada para Orphans + Filtro por Status

Annotations são agrupadas por anchor status. Orphans ficam em seção colapsável no final. Filtro novo por status.

Banner com contagem clicável que filtra. Highlights no conteúdo mudam de cor por status. Sem mudança estrutural no painel.

Contexto Técnico

reanchorAll() já retorna Map<id, {status: 'position'|'exact'|'fuzzy'|'orphan'}>.
O gap é surfacear esse resultado na UI. A issue propõe mapear:
position/exact → anchored · fuzzy → drifted · orphan → orphaned
